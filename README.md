# AWS OTT Platform

A Netflix-style OTT platform hosted on Amazon EC2 that streams movie trailers stored in Amazon S3.

## Features

- Responsive OTT interface
- Movie posters from Amazon S3
- Video trailers from Amazon S3
- Hosted on Amazon EC2
- HTML, CSS and JavaScript

## AWS Services

- Amazon EC2
- Amazon S3
- IAM
